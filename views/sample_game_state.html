Sample Preliminary Software Design

    
Model

    
Constants

    EASY_BOARD the grid containing the given numbers for the corresponding easy difficulty level.
    MEDIUM_BOARD the grid containing the given numbers for the corresponding medium difficulty level.
    HARD_BOARD the grid containing the given numbers for the corresponding hard difficulty level.

    
Objects

    
userInput

    Each user guess object will have a selected cell and a rank property.
    Selected call will be one of the following: grid[0-8][0-8]
    User input will be a number 1-9.
    User input object examples:
    {selectedCell: [0,3], input: 7};
    {selectedCell: [3,6], input: 1};
    {selectedCell: [8,0], input: 4};

    
Functions/methods

    setTime() : Creates a timer for the game.
    getDifficulty() : Retrieves the difficulty level set by the player
    checkAnswer() : Checks user input against the answer key, checking if the input is correct.
    useHint() : Checks to see if the user has hints to use and then reveals the next cell.
    pauseGame() : Pauses the timer and hides the game grid.
    resumeGame() : Resumes the game. Starting the timer and revealing the board.
